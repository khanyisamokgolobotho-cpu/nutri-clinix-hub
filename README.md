# Nutri-Clinix Hub

Professional React + Vite + Tailwind CSS website mockup for a dietetics and nutrition practice.

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## GitHub Pages

The site deploys through GitHub Actions when changes are pushed to `main`.
